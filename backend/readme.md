1. Импорт библиотек:
```
# для запуска сессии, в которой будут храниться данные для сервера
import threading

# для http-запросов
import requests

# для передачи питон-программе аргументов
# будем передавать конфиг-файл
import argparse
```
2. Создание класса Server для перехвата запросов (API клиент-сервер);
3. Парсинг конфиг-файла и передача данных конфигураци и в объект-сервер.


## 2. Настройка базы данных (API сервер-БД).

1. Создание класса MySQLConnection, определение прав доступа и сессии;
2. В MySQLConnection создаем движок ORM (Object-Relational Mapping) - позволяет взаимодействовать с БД не через язык запросов SQL, а в ООП-формате;
3. Создание класса DbInteraction (API сервер-БД);
